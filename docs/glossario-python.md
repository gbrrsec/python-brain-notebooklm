# Glossário Python

Este glossário reúne os 50 conceitos mais importantes da linguagem Python, abrangendo fundamentos, tipos de dados, estruturas de controle, funções e programação orientada a objetos.

---

# Fundamentos e Sintaxe

## Python
Linguagem de programação de alto nível, interpretada, multiparadigma e conhecida pela simplicidade e legibilidade.

## Interpretador
Programa responsável por executar o código Python linha a linha.

## Indentação
Uso de espaços em branco para definir blocos de código. Em Python, a indentação é obrigatória.

## Variável
Espaço reservado na memória utilizado para armazenar dados.

## Tipagem Dinâmica
Permite que o tipo de uma variável seja definido automaticamente durante a execução.

## Tipagem Forte
Impede operações entre tipos incompatíveis sem conversão explícita.

## Case Sensitive
Diferencia letras maiúsculas de minúsculas. Exemplo: `Nome` e `nome` são variáveis diferentes.

## Comentário
Texto utilizado para documentar o código. Em Python, utiliza-se `#`.

## Identificador
Nome atribuído a variáveis, funções, classes ou módulos.

## Palavras Reservadas
Termos próprios da linguagem que não podem ser utilizados como identificadores, como `if`, `for` e `def`.

---

# Tipos de Dados e Coleções

## int
Tipo de dado utilizado para números inteiros.

## float
Tipo de dado utilizado para números com casas decimais.

## str (String)
Representa uma sequência de caracteres ou texto.

## bool (Boolean)
Representa valores lógicos: `True` ou `False`.

## None
Representa a ausência de valor.

## list (Lista)
Coleção mutável, ordenada e indexada.

## tuple (Tupla)
Coleção ordenada e imutável.

## dict (Dicionário)
Estrutura que armazena dados em pares de chave e valor.

## set (Conjunto)
Coleção não ordenada que não permite elementos duplicados.

## Mutabilidade
Capacidade de um objeto ser alterado após sua criação.

## Imutabilidade
Característica de objetos que não podem ser modificados após serem criados.

## Índice
Posição numérica de um elemento dentro de uma sequência.

---

# Estruturas de Controle e Operadores

## Operador de Atribuição (=)
Utilizado para atribuir valores a variáveis.

## Operador Módulo (%)
Retorna o resto de uma divisão inteira.

## Concatenação
União de duas ou mais strings utilizando o operador `+`.

## if / elif / else
Estruturas condicionais utilizadas para tomada de decisão.

## while
Laço de repetição executado enquanto uma condição for verdadeira.

## for
Laço utilizado para percorrer sequências e coleções.

## break
Interrompe imediatamente a execução de um laço.

## continue
Interrompe apenas a iteração atual e avança para a próxima.

## pass
Instrução utilizada como marcador quando nenhum código precisa ser executado.

## range()
Função que gera sequências numéricas.

---

# Funções e Modularização

## def
Palavra-chave utilizada para definir funções.

## return
Retorna um valor e encerra a execução da função.

## lambda
Função anônima escrita em uma única linha.

## *args e **kwargs
Permitem receber uma quantidade variável de argumentos em funções.

## Docstring
Texto de documentação utilizado para descrever funções, métodos e classes.

## Recursividade
Técnica em que uma função chama a si mesma.

## Módulo
Arquivo Python (`.py`) reutilizável em outros projetos.

## Pacote
Conjunto de módulos organizados em diretórios.

## PIP
Gerenciador de pacotes oficial do Python.

---

# Programação Orientada a Objetos e Tratamento de Erros

## Classe
Estrutura utilizada como modelo para criação de objetos.

## Objeto
Instância criada a partir de uma classe.

## Atributo
Característica ou propriedade associada a um objeto.

## Método
Função definida dentro de uma classe.

## self
Referência ao próprio objeto dentro de seus métodos.

## __init__
Método construtor executado durante a criação de um objeto.

## Exceção (Exception)
Erro detectado durante a execução do programa.

## try / except
Estrutura utilizada para capturar e tratar erros.

## Traceback
Relatório exibido quando ocorre um erro, indicando onde ele aconteceu.

---

# Conclusão

Dominar esses conceitos é fundamental para compreender a linguagem Python e avançar para tópicos mais complexos, como desenvolvimento web, automação, ciência de dados, inteligência artificial e desenvolvimento de software profissional.
