# Plano de Estudos de Python - 30 Dias

## Objetivo

Este plano foi desenvolvido para ajudar iniciantes a aprender Python em 30 dias, combinando teoria e prática de forma progressiva. Recomenda-se dedicar aproximadamente 2 horas por dia aos estudos.

---

# Semana 1 - Configuração e Fundamentos

## Dias 1 e 2 - Instalação e Ambiente

### Objetivos
- Instalar o Python 3.
- Configurar um ambiente de desenvolvimento.
- Executar o primeiro programa.

### Atividades
- Instalar o Python.
- Configurar VS Code ou PyCharm.
- Criar o programa "Hello World".

---

## Dias 3 a 5 - Variáveis e Tipos de Dados

### Objetivos
- Compreender como armazenar informações.
- Conhecer os principais tipos de dados.

### Conteúdos
- Variáveis.
- int.
- float.
- str.
- bool.

### Exercícios
- Criar programas utilizando entrada de dados com `input()`.
- Exibir resultados com `print()`.

---

## Dias 6 e 7 - Operadores

### Conteúdos
- Operadores aritméticos.
- Operadores relacionais.
- Operadores lógicos.

### Exercícios
- Criar calculadoras simples.
- Realizar comparações entre valores.

---

- Sistema simples de login.

### Conteúdos

### Exercícios

- Soma de números em sequência.

---

## Dia 14 - Revisão

### Atividades

### Projetos Simples

# Recomendações

- Pratique todos os dias.
- Digite os códigos manualmente.
- Aprenda com os erros.
- Consulte a documentação oficial sempre que possível.
- Utilize o GitHub para armazenar seus projetos e acompanhar sua evolução.

---

# Resultado Esperado

Ao final dos 30 dias, o estudante será capaz de:

- Compreender a sintaxe básica do Python.
- Criar programas simples e intermediários.
- Utilizar estruturas de dados.
- Desenvolver funções reutilizáveis.
- Aplicar conceitos de Programação Orientada a Objetos.
- Construir projetos práticos para compor seu portfólio.
