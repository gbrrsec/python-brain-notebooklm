# Critérios de Seleção

As fontes foram escolhidas com base nos seguintes critérios:

- Conteúdo atualizado.
- Material voltado para iniciantes e intermediários.
- Fontes reconhecidas pela comunidade de programação.
- Cobertura de conceitos fundamentais e avançados da linguagem Python.
- Disponibilidade gratuita ou de livre acesso para consulta.

# Aplicação no Projeto

Todas as fontes listadas foram utilizadas como base de conhecimento dentro do NotebookLM para:

- Geração de resumos.
- Criação de glossário técnico.
- Elaboração de plano de estudos.
- Geração de exercícios.
- Simulação de entrevistas técnicas.
- Correção e análise de código Python.
